XcelDB
======

Keystore written in GO with Raft consensus



Features
====

* Distributed
* Strongly consistent
* Highly Available

Requirements
============
* Need alteast go1.1 or newer
* Need Raft
(see usage for more info)

Usage
===== 
* Install Raft from github.com/nilangshah/Raft (see usage instruction of Raft)
* Run tests to test XcelDB 
 * Make
